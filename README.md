# Demyx discord bot
Bot de discord voltado para musica construido usando Javascript e a biblioteca discord.js.
# Funcionalidades implementadas
!play : recebe um link do youtube e começa a tocar o audio do video no canal de voz, e se ja estiver tocando adiciona uma musica para o final da fila de reprodução. \n
!skip : pula para a proxima musica da lista. \n
!stop : desconecta o bot do servidor e limpa a lista de reprodução. \n
